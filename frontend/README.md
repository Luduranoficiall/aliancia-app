# Frontend — ALIANCI.A

Next.js 14, TailwindCSS, PWA, painel do membro, site institucional premium.

## Estrutura
- `app/`: páginas principais
- `components/`: componentes reutilizáveis
- `styles/`: Tailwind e customizações
- `public/`: assets

## Como rodar
```bash
cd frontend
npm install
npm run dev
```

## Deploy
- Vercel / Fly.io
